## 🎨 颜色应用参考

### 背景颜色层级
```
┌─ 极深黑 (#050505)
│  └─ 用途: 页面主背景、文本背景
│
├─ 深蓝黑 (#0B1220)
│  └─ 用途: 模态框、弹窗背景
│
└─ 玻璃层 (white/5)
   └─ 用途: 卡片背景、表格背景 (带blur)
```

### 文本颜色梯度
```
#F8FAFC (最亮)  ← 主标题、关键数字
   ↓
#E2E8F0 (亮)   ← 主文本、标签
   ↓
#94A3B8 (中)   ← 副文本、时间戳
   ↓
#475569 (暗)   ← 禁用状态、提示文本
```

### 状态颜色映射
```
成功 #10B981 ────────┬──→ 绿色圆点 ✓
                     └──→ 状态徽章 "成功"

警告 #F59E0B ────────┬──→ 橙色圆点 ⏳ (脉冲)
                     └──→ 状态徽章 "待处理"

失败 #EF4444 ────────┬──→ 红色X号 ✗
                     └──→ 状态徽章 "失败"

数据 #22D3EE ────────┬──→ 批次Tag背景
                     └──→ 按钮高亮
```

## 🖱️ 交互状态示意

### 复制按钮状态流
```
【鼠标离开】                【鼠标点击】          【2秒后】
   灰色                      变绿✓              恢复灰色
    ⬜                       ✅                   ⬜
    
【用户感受】
"哦，复制成功了！"
```

### 批次Tag交互流
```
【正常显示】               【鼠标悬停】              【点击】
蓝色Tag (Batch:...)      背景变深蓝              表格重新过滤
点击区域 cursor-pointer   cursor-pointer          出现"清除筛选"

【结果】表格仅显示同批次2条
```

### Tooltip交互流
```
【正常状态】
红色⚠️ 图标
（无提示）

【鼠标悬停】
红色⚠️ 图标
   ↓
【显示Tooltip】
"High-risk address detected 
 in whitelist violation"
（带红色背景和小箭头指向）

【鼠标离开】
Tooltip消失
回到正常状态
```

## 🎭 动画时序

### 过渡动画 (Smooth 200ms)
```
悬停前 ─── 200ms过渡 ──→ 悬停后

border-white/10  white/20
bg-transparent   bg-white/5
text-gray-400    text-white

效果: 平滑亮化，无闪烁
```

### 脉冲动画 (Pending状态)
```
初始态     延迟     50%         延迟     100%
   |       |        |          |        |
   ●       →      ●●●●●   →    ●       ← 循环
(小)           (大+淡)      (小)

持续: 2s, 无限循环
```

### 复制反馈动画
```
点击
  ↓ (立即)
✓ 显示 (绿色)
  ↓ (2秒后)
⬜ 恢复 (灰色)
```

## ✨ 视觉亮点总结

| 特点 | 效果 | 用户感受 |
|------|------|--------|
| Glass-morphism | 卡片有景深感 | "科技感强，专业" |
| 深黑OLED主题 | 省电、护眼 | "舒适、高端" |
| 彩色状态指示 | 一目了然 | "清晰、直观" |
| 微交互反馈 | 每个操作有反应 | "流畅、可控" |
| 平滑过渡 | 无闪烁跳动 | "稳定、可信" |
| 高对比度文本 | 易于阅读 | "清晰、易读" |
| 响应式设计 | 任何设备都好用 | "贴心、周到" |
| 图标 + 文字 | 双重标识 | "无歧义、安心" |

## 🚀 优化建议

1. **性能**: 表格虚拟化 (>100条数据时)
2. **功能**: 添加排序、导出、详情侧边栏
3. **体验**: 添加"加载中"状态、骨架屏
4. **安全**: 地址脱敏显示 (0x7...f42e1)
5. **国际化**: 支持中英双语切换
