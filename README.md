<h1 align="center">TFx402</h1>

TFx402 是一个基于 Tron 的、面向 AI Agent 的稳定币支付平台，支持 x402 协议。它专注于低手续费转账、严格的资金安全管控，以及可靠的实时处理能力。

## 亮点

- Tron 网络手续费低
- 原生支持 x402 协议
- 账户资金安全管控
- 安全、实时的支付处理
- 支持交易历史查询与审计
